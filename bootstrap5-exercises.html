19. Customization with Sass

Exercise 19.1 & 19.2: Sass Customization

To set up Bootstrap with Sass customization:

  1. Install Bootstrap via npm: npm install bootstrap
  2. Create a custom SCSS file importing Bootstrap variables
  3. Override variables like $primary and $border-radius
  4. Import Bootstrap's main SCSS file
  5. Compile using a Sass compiler
